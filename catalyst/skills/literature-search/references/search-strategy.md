# Search Strategy

## Search Rules

1. Start with the narrowest query that still matches the research objective.
2. Expand only when results are too sparse or clearly off target.
3. Record source quality, not just source quantity.

## Evidence Rules

1. Prefer concise paraphrases over long quotations.
2. Preserve source identifiers so evidence can be traced later.
3. Separate factual findings from inferred conclusions.
